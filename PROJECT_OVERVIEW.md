# Tata BarberShop Project Overview

## Project

Tata BarberShop is a static barbershop website adapted from a WordPress/Elementor export. The current site is focused on online booking, local business information, services, pricing, gallery, contact details and hero content management.

## Main Entry Points

- `index.html`: public website.
- `admin.html`: hero/admin content editor.
- `static/css/tata-ui-polish.css`: custom layout, spacing, responsive and brand polish.
- `static/js/tata-ui-polish.js`: custom navigation, booking links, contact data, logo injection and UI enhancements.
- `static/js/tata-language.js`: frontend language switcher.
- `static/js/tata-baber-supabase-config.js`: Supabase/admin fallback data for hero content.
- `static/js/tata-baber-hero-loader.js`: loads hero settings/slides from Supabase or local defaults.
- `static/js/tata-baber-about-loader.js`: loads and renders WHO WE ARE images from Supabase or local defaults.
- `hero_schema.sql`: Supabase schema for hero settings, hero slides and about images.

## Admin Notes

The admin page is branded as Tata BarberShop Admin. Hero content can be loaded from Supabase when configured, with local defaults available as a fallback.

Admin modules:

- Hero Settings
- Hero Slides
- About Images

For production, avoid exposing Supabase service-role keys in public frontend files. Use an API/backend or Supabase policies with anon-safe access instead.
